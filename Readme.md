# Google Colab Extension for VS Code Guide

This guide explains how to use the Google Colab extension in VS Code to connect local notebooks to powerful Colab runtimes.

## ðŸš€ Getting Started

1.  **Install the Extension**:
    *   Open the Extensions view in VS Code (`Ctrl+Shift+X`).
    *   Search for **Google Colab**.
    *   Click **Install** on the official extension.

2.  **Connect to a Runtime**:
    *   Open or create a `.ipynb` notebook file.
    *   Click the **Select Kernel** button (top right) or run a cell.
    *   Select **Colab** from the options.
    *   Choose your desired runtime and sign in with your Google account.

Your local notebook is now powered by Google Colab!
